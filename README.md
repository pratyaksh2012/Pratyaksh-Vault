# ☁️ Pratyaksh -Vault 

 Pratyaksh -Vault is a secure cloud-based file sharing web application built using Firebase.

## Features

- User Login & Signup
- File Upload
- My Files
- File Preview
- File Download
- Share Files with Other Users
- Shared Files
- Delete Files
- Search Files
- Dashboard
- Dark Mode
- Responsive UI

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Firebase Authentication
- Firebase Firestore
- Firebase Storage
